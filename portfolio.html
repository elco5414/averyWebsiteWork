Silver Hollow

Silver Hollow is a command line game designed by myself and my partner for the HACKCU9 hackathon in the spring of 2023. The game features choices for the user, a map feature, an inventory that updates itself, and demonstrates a thorough understanding of the C++ computing language. The project won its track in the Hackathon and can be veiwed on both github and devpost, linked below.
